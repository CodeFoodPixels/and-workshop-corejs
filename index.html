Core Javascript

Learn JavaScript in an applicable way through re-implementing the main libraries.

This course has been designed with 🌴 remote in mind and intentionally created with the ability for you to re-visit the content (and hopefully teach it to others!).

The course is centred around mimicking popular libraries and JavaScript API's, this is to de-mystify design patterns and concepts.

Ready to rock & roll? 🎸 🍞 — let's get to it!

Disclaimer:

This course covers only core JS language, we will not cover execution environments (no DOM or Node APIs)

Getting Started:

  • Install Jest npm install jest --global
  • Install Quokka JS. to understand the files easier
  • Update your exercises with your code
  • Write comments to explain what you've done

Trainer Instructions:

  • Read through description parts thoroughly — but quickly.
  • Experiment live with examples as you go, explain, discuss.
  • Start on exercises as pairs, swapping frequently
  • At the end: Feedback / share / discuss as a group.
  • If complete, extend, improve, re-write answers in alternative flavours / styles, bolster test coverage.

Remember: Try to spend most of time on the exercises themselves and discussing real worked examples
